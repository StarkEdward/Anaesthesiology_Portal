# बधिरीकरण विभाग साहित्य व्यवस्थापन (Anesthesia Inventory) - Local Setup Guide

तुमच्या Windows 11 PC वर हे ॲप ऑफलाइन चालवण्यासाठी खालील पायऱ्यांचे अनुसरण करा:

## १. आवश्यक सॉफ्टवेअर (Prerequisites)

हे ॲप चालवण्यासाठी तुमच्याकडे **Node.js** असणे आवश्यक आहे.
- [nodejs.org](https://nodejs.org/) वर जा.
- **LTS (Recommended for Most Users)** व्हर्जन डाउनलोड करा आणि इन्स्टॉल करा.

## २. प्रोजेक्ट फाइल्स मिळवणे (Getting the Files)

- AI Studio मधील **Settings** (वरच्या उजव्या कोपऱ्यात) वर क्लिक करा.
- **Export to ZIP** निवडा.
- तुमच्या PC वर ती ZIP फाईल अनझिप (Extract) करा.

## ३. लायब्ररी इन्स्टॉल करणे (Installing Dependencies)

- अनझिप केलेल्या फोल्डरमध्ये जा.
- वरच्या ॲड्रेस बारमध्ये `cmd` टाईप करा आणि Enter दाबा (यामुळे कमांड प्रॉम्प्ट उघडेल).
- खालील कमांड टाईप करा आणि Enter दाबा:
  ```bash
  npm install
  ```

### ⚠️ महत्त्वाचे (Firebase Setup):
हे ॲप आता Firebase वापरत आहे. ॲप सुरू करण्यापूर्वी तुमच्याकडे `firebase-applet-config.json` असणे आवश्यक आहे किंवा तुम्ही तुमचे स्वतःचे Firebase प्रकल्प जोडू शकता. 
Firebase चा वापर केल्यामुळे इंटरनेट कनेक्शन आवश्यक आहे आणि सर्व डेटा सुरक्षितपणे क्लाउड (Cloud) मध्ये सेव्ह होतो.

## ४. ॲप सुरू करणे (Starting the App)

- कमांड प्रॉम्प्टमध्ये खालील कमांड टाका:
  ```bash
  npm run dev
  ```
- आता तुमचा ब्राउझर (Chrome/Edge) उघडा आणि `http://localhost:3000` किंवा `http://४०००` (जर पोर्ट बदलला असेल) वर जा.

## ५. वापर (Usage)

- एकदा `npm install` पूर्ण झाले की, ॲप सुरू करण्यासाठी इंटरनेटची गरज आहे.
- तुमचा डेटा Firebase मध्ये सुरक्षित राहील.

---

### काही महत्त्वाच्या टिप्स:
- **डेटा बॅकअप**: तुमचा डेटाबेस Firebase वर आहे, जो स्वतः बॅकअप घेतो.
- **प्रिंटिंग**: पावती प्रिंट करण्यासाठी तुमच्या PC ला प्रिंटर जोडलेला असावा.
- **बंद करणे**: कमांड प्रॉम्प्टमध्ये `Ctrl + C` दाबून तुम्ही ॲप बंद करू शकता.
